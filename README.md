# COSC_4301 Spring 2026 – Coursework Repository

This repository contains projects and exercises completed as part of a Spring 2026 software development course.

## Purpose

This course is being taken independently as part of ongoing professional development while pursuing entry-level software engineering opportunities. The goal is to continue strengthening backend development skills, particularly in:

* Java & Spring Boot
* REST API design
* Database integration (PostgreSQL, JPA/Hibernate)
* Debugging and troubleshooting real-world issues

## Projects Included

* **neon-ark-intake-tracker**
  A Spring Boot application implementing CRUD operations with DTOs, validation, and relational database integration.

* **my-docker-java-monster**
  A containerized Java application demonstrating Docker fundamentals and deployment workflows.

## Notes

This repository reflects active skill-building and hands-on practice. It includes working code, troubleshooting notes, and iterative improvements made throughout the course.
